# 🛍️ Customer Behavior Analysis Dashboard

An end-to-end Data Analytics project that analyzes customer purchasing behavior using **Python, PostgreSQL, SQL, and Power BI**.

The project demonstrates the complete data analytics workflow—from data cleaning and preprocessing to database management, SQL analysis, and interactive dashboard creation.

---

## 📌 Project Overview

Businesses collect large amounts of customer data, but extracting meaningful insights from it is challenging.

This project focuses on analyzing customer purchasing patterns to answer important business questions such as:

- Which gender contributes more revenue?
- Which product categories generate the highest sales?
- Which age groups spend the most?
- Which seasons produce maximum sales?
- How does customer behavior vary across different categories?

The final insights are presented through an interactive Power BI dashboard.

## 📊 Project Workflow

### 1. Data Collection

### 2. Data Cleaning (Python)

- Data type conversion
- Feature formatting
- Export cleaned dataset

### 3. Database

- Created PostgreSQL database
- Imported cleaned dataset into PostgreSQL

### 4. SQL Analysis

Performed analytical queries including:

- Revenue by Gender
- Revenue by Product Category
- Customer Count by Gender
- Average Purchase Amount

- Product Category Analysis

### 5. Dashboard

- Revenue Analysis

- Sales by Category
